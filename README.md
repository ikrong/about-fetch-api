# about fetch API

这是一个关于 fetch API 的简介，详细介绍 fetch API 及在 ChatGPT 中的应用。

# 目录结构

```
src/              源代码
  client/         html静态页面
    index.html    主页面
    style/        样式
    script/       脚本
    md/           文档
    libs/         第三方库
    assets/       静态文件
  server/         nodejs服务
```

# 启动

```sh
npm start
npm run css # 如果需要更改样式，可以启动这条命令
```
